# Create Feature Preview

Add a new feature preview to the gonfalon codebase using the plop generator.

## Quick Start

```bash
cd /path/to/gonfalon
pnpm create:feature-preview
```

## When to Use

Use this skill when you need to:
- Add a new opt-in experimental feature to the LaunchDarkly UI
- Create a feature preview that users can enable from their settings
- Set up a gated feature with feedback collection
